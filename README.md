# GREENIFY
GREENIFY is a gamified environmental education platform that transforms theoretical learning into real-world action. Through mini-games, interactive lessons, and a 3D Unity simulation, students learn sustainability, track eco-points, and build lasting green habits.


GREENIFY is a gamified environmental education platform that transforms textbook-based learning into interactive, action-driven experiences. It combines mini-games, real-world challenges, and a 3D Unity simulation game to help students understand the impact of their daily choices on the environment.
